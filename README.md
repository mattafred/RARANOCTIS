# Rara Noctis

Prototype front-end autonome d’une boutique de bijoux de corps haut de gamme.

## Ouvrir le site

Ouvrez directement `index.html` dans un navigateur moderne. Aucune installation ni dépendance n’est nécessaire.

## Fonctionnalités

- page d’accueil responsive ;
- catalogue de huit produits ;
- fiches produit avec matières, dimensions, entretien et livraison ;
- panier avec quantités et sous-total en dinars tunisiens ;
- commande préparée directement dans WhatsApp ;
- favoris persistants ;
- recherche instantanée ;
- menu mobile et animations respectant `prefers-reduced-motion`.

## Avant une mise en production

Les illustrations vectorielles sont des visuels de direction artistique. Elles doivent être remplacées par les photographies et vues portées de la marque. Il faudra également connecter le catalogue, le stock, les comptes clients, le paiement, la livraison, les e-mails transactionnels, les mentions légales et la mesure d’audience à une vraie plateforme e-commerce.
